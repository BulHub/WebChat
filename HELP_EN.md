# Application Download and Launch Guide

## Download

1. Latest working version:
https://github.com/BulHub/WebChat/releases/tag/v1.0.0
2. Download zip archive

## Running

1. Open a project with a Java-enabled IDE
2. Create tables: BulChat\src\main\webapp\data.sql
3. Replace data for connecting to the database:
BulChat\src\main\java\ru\bulat\data\DatabaseConnection.java
4. Start chat server using TomCat
